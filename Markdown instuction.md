# Инструкция для работы с Markdown

##  Выделение текста

Чтобы выделить текс курсивом, необходимо его обромить звездочками (*) Например

*текст*

Чтобы выделить текс жирным, необходимо его обромить двумя звездочками (**) Например

**текст**
